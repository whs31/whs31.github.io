Detailed Description

Class for working with local application settings, stored in filesystem.

Provides access to reading and writing application settings to local .ini file. Automatically load default settings on fresh application install. Class mainly used as wrapper over QSettings instance, providing useful invokable functions and properties avialable in QML.

Config class also provides useful macro for accessing it's properties from different classes and namespaces:

CONFIG(property_name);

Example usage of macro:

Network::startTelemetrySocket(CONFIG(remoteIP) + ":" + CONFIG(telemetryPort), CONFIG(telemetryFrequency));

angleCorrection
readwrite

Debug property for adding or subtracting angle values from image metadata.

Value describes, how many degrees will be added/subtracted from image metadata azimuth. By default, this property returns 0 (no correction).

Note
In current software version, it is probably better approach to use image editor for individual images. Setting this property to any values other than 0 can result in strange image geopositioning behaviour.

◆ useRadians

readwrite

Compatibility setting for older firmware versions.

If boolean evalueates to true, image metadata decoding process will expect radian values in azimuth, drift angle and div fields. VT45-firmware (target platform) currently behaves like that.
