struct QuasarSDK::Datagrams::PlannerAppendDatagram

Структура добавления точки в полётное задание.

Для создания полетного задания необходимо передать сервису сообщение, которое состоит из заголовка и данных. Данные должны располагаться сразу же после заголовка. Данные должны быть представлены в виде строки в формате execd.

Пример:

// для этой команды len = 30
$some_script(arg1, arg2, arg3)
Заметки
Сериализована в Big Endian.

◆ QuasarSDK::Datagrams::PlannerRemoveDatagram

struct QuasarSDK::Datagrams::PlannerRemoveDatagram

Структура удаления точки из полётного задания.

Точки удаляются автоматически при достижении cnt = 0; При очередном достижении зоны события счетчик cnt уменьшается на 1. Для ручного удаления точки необходимо передать PlannerRemoveDatagram.

Заметки
Сериализована в Big Endian.

struct QuasarSDK::Datagrams::PlannerResponseDatagram

Структура ответа на прохождение БПЛА через точку из полётного задания.

При достижении зоны события сервис отправляет сообщение по адресу, указанному при добавлении точки.
