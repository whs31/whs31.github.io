uint16_t marker = 0 Идентификатор протокола.

Равен 0xDEFA.

uint16_t version = 0 Версия протокола.

По маске 0xF000 - мажорная, по 0x0FFF - минорная.

uint16_t size = 0 Размер пакета в байтах, включая заголовок.
uint16_t cnt = 0 Счётчик.
uint16_t id = 0 Идентификатор РЛС.

0x55 - РЛС "Квазар".

uint16_t type = 0 Идентификатор типа сообщения.

0x1 - полосовое РЛИ. 0x2 - таблица обнаружений.
