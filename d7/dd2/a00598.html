Подробное описание

Тип данных для работы с сервисом execd.

ExecdArgument предоставляет полезные конструкторы и функции для абстракции над аргументами сервиса. Вместо приведения различных типов данных к строке вручную, используйте этот тип.

Заметки
После вызова конструктора тип аргумента не может быть изменен. Инициализируйте аргументы значениями по умолчанию, а не пустыми конструкторами.
