Detailed Description

Класс-синглтон для работы с локальными настройками приложения, сохранямыми на диске.

Класс предоставляет доступ к чтению и записи настроек конфигурационного .ini файла приложения. Автоматически загружает настройки по умолчанию на чистой установке приложения. Основное предназначение класса - обёртка над экземпляром класса QSettings, предоставляющая глобально доступные функции и свойства (в том числе для QML).

Класс Config также предоставляет полезный макрос для доступа к его свойствам и функциям из других классов и пространств имён в С++:

CONFIG(property_name);

Пример доступа к конфигурационному файлу приложения из С++:

Network::startTelemetrySocket(CONFIG(remoteIP) + ":" + CONFIG(telemetryPort), CONFIG(telemetryFrequency));
