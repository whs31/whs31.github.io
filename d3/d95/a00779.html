Подробное описание

Класс для приема больших файлов с РЛС по протоколу TCP-IP.

Используется для приема цельных пакетов с телескопическими и полосовыми изображениями с РЛС, а также для приёма .zip-архивов.

Предоставляет отладочные метрики по аналогии с классами-наследниками BaseUDPSocket.
