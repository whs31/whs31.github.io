Подробное описание

Датаграмма телеметрии от РЛС к наземке.

Датаграмма приходит в ответ на запрос к РЛС. Исходная датаграмма в Rust выглядит так:

#[repr(C, packed)]
#[derive(Serialize)]
pub struct Datagram
{
marker : u32,
version : u8,
lat : f64,
lon : f64,
ele : f64,
velocity_course : f64,
velocity_east : f64,
velocity_north : f64,
velocity_vertical : f64,
pitch : f64,
roll : f64,
yaw : f64,
course : f64,
time : u64,
valid : bool,
crc : u16
}
