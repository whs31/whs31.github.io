Подробное описание

Class for receiving feedback from SAR.

Class is used to receive stdout and stderr output from SAR, redirected from execd service.

Output can contain VT100 escape sequences, which needs to be parsed.

Class automatically parse predefined string sequences (e.g. free storage space status) and emits signal respectively.

Provides debug metrics.
