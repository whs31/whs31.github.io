Подробное описание

Класс-синглтон, отвечающий за связь с РЛС.

Фабричный класс, регулирующий инстанциирование и запуск объектов сокетов TCP и UDP. Класс предоставляет функции для чтения и записи аргументов формирования, проверки доступности соединения и прямого управления сокетами. Также предоставляет указатели на хранилища данных (телеметрии и общих данных РЛС). Для отладочной информации класс предоставляет в виде сигналов метрики каждого дочернего сокета.

Заметки
Класс необходимо зарегистрировать как singletonInstance для доступа к нему из QML.

Производит попытку подключения к РЛС.

Функция запускает все дочерние сокеты с выбранными параметрами и адресами.

Аргументы
telemetry_request_addr- IP-адрес для запросов телеметрии в формате 192.168.1.0:5555.
telemetry_recv_addr- IP-адрес для получения телеметрии.
telemetry_frequency- интервал в секундах между пакетами телеметрии.
execd_addr- IP-адрес для команд в сервис Execd на РЛС.
feedback_addr- IP-адрес для приема потока stdout с РЛС.
tcp_lfs_addr- IP-адрес для сервера TCP-IP.
udp_lfs_addr- IP-адрес для сокета полосовых изображений.
Заметки
Может быть вызвана из QML через мета-объектную систему.
